# Heritage & Haute Boutique Website

## Overview

**Heritage & Haute** is a modern, luxury boutique website developed using **React.js** and **Vite**. The website is designed to showcase premium fashion collections through an elegant and responsive user interface. It provides visitors with an immersive browsing experience by highlighting the brand story, curated collections, signature styles, customer testimonials, and an interactive lookbook.

---

* Modern luxury-inspired UI
